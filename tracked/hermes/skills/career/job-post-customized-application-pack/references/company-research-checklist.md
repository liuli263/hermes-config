# 加拿大公司求职调研清单（AI / Hermes 执行版）

适用场景：收到公司名或职位 JD 后，先做公司侧调研，再做投递判断与定制化材料生成。

## 1. 基础信息
- 公司全称
- 官网链接
- LinkedIn 公司主页
- 总部所在地
- 加拿大办公地点（尤其 Montreal / West Island / Kirkland 附近）
- 公司规模（全球 / 加拿大）
- 行业 / 主营业务
- 成立时间
- 是否上市、股票代码
- 母公司 / 集团关系
- 近一年业务动态：融资、裁员、并购、扩张、重大负面新闻

## 2. 招聘与岗位情况
- 加拿大是否正在招聘
- 常见招聘城市
- 常见岗位类别
- 工作模式：Remote / Hybrid / Onsite
- 是否支持工签 / LMIA / sponsorship
- 招聘页面链接
- 招聘频率是否稳定
- LinkedIn 近期职位数量

## 3. 员工评价与口碑
优先来源：Glassdoor、Indeed、Reddit、Blind、LinkedIn 公开资料。

采集：
- Glassdoor / Indeed 评分
- CEO approval / Recommend to a friend
- 高频优点：氛围、WLB、稳定性、学习机会、薪资福利
- 高频缺点：加班、政治、晋升慢、裁员、管理混乱
- 员工流动性 / 任职年限偏短迹象
- Reddit / 论坛真实吐槽关键词

## 4. 薪资情况
优先加拿大数据：
- 岗位名称
- 城市
- Base salary
- Bonus
- Equity / RSU / Options
- Total compensation
- 来源（Glassdoor / Indeed / Levels / LinkedIn salary 等）

## 5. 福利待遇
标注为：已确认 / 未确认 / 员工提及但未官方确认
- Extended Health
- Dental
- Vision
- Mental health support
- RRSP matching / Pension
- Bonus
- Stock / RSU / ESOP
- Vacation days
- Sick leave
- Parental leave
- Learning budget
- Fitness
- Remote stipend
- Employee discount
- Flexible schedule

## 6. 面试流程
- 回复速度
- 面试轮次
- HR screening / hiring manager / technical / panel / final
- 难度
- 常见问题类型
- 是否有 take-home
- 是否有 background check / reference check

## 7. 风险排查
- 近一年裁员
- 劳动纠纷 / 集体诉讼
- 大量负评
- 管理层频繁变动
- 财务不稳定
- 高频差评关键词：高压、内耗、频繁重组、offer 撤回、冻结招聘
- 加拿大实体是否真实存在

## 8. 与用户偏好的匹配
对当前用户至少额外判断：
- 是否在 Kirkland 车程 30 分钟内，若无精确地图则至少根据 office 地址做地理初筛
- 是否时薪 >= 20 CAD（年薪按 2080 小时折算）
- 英语是否优先 / 工作语言是否友好
- 是否 remote / hybrid
- 年假和福利是否偏好向

## 9. 推荐搜索关键词
- `{公司名} Canada careers`
- `{公司名} LinkedIn company`
- `{公司名} Glassdoor`
- `{公司名} Indeed reviews`
- `{公司名} salary Canada`
- `{公司名} benefits`
- `{公司名} Reddit`
- `{公司名} layoffs`
- `{公司名} interview Glassdoor`
- `site:linkedin.com/company {公司名}`
- `site:reddit.com {公司名}`
- `site:glassdoor.ca {公司名}`
- `site:indeed.com/cmp {公司名}`

## 10. 建议输出结构
1. 一页摘要
2. 详细报告
3. 表格汇总（信息项 / 结果 / 来源 / 可信度）
4. 来源清单（官网 / LinkedIn / Glassdoor / Indeed / Reddit / 新闻 / 企业注册）
